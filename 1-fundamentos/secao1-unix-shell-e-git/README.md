# Seção 1 - UNIX, SHELL E GIT

Esse bloco pertence ao módulo de [`fundamentos`](/1-fundamentos) do curso da [Trybe](https://www.betrybe.com/).

Nesse bloco aprendi sobre:

- O que são sistemas baseados em Unix
- Os principais comandos utilizados no Shell: navegar, criar pastas, criar e editar arquivos, etc.
- Os principais comando do Git para versionamento local e remoto utilizando o GitHub

Tendo isso em mente aqui você encontrará os exercícios que desenvolvi para fixar os conhecimentos que adquiri ao longo desse bloco.
